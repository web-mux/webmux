## 🧠 Project Overview

**Webmux** — bu zamonaviy va jonli foydalanuvchi interfeyslariga e’tibor qaratilgan professional front-end loyihadir.  
U intuitiv dizayn, ko‘p animatsiyalar va interaktivlik orqali foydalanuvchilarga unutilmas tajriba taqdim etadi.

🎯 Maqsadim — nafaqat ishlaydigan, balki his qilinadigan saytlar yaratish.

---

## 🌈 Nimalar ishlatilgan?

### 🧩 Texnologiyalar:
- **HTML5** – semantik tuzilma
- **CSS3** – Grid, Flexbox, media queries
- **JavaScript (ES6+)** – logika va interaktivlik
- **React.js** – komponent asosli arxitektura

### ✨ Animatsiyalar:
- `Framer Motion` – React uchun eng zamonaviy animatsiyalar
- `CSS Animations` – hover, fade, slide effektlar
- `Scroll Reveal` – sahifa scroll bo‘lganda elementlar paydo bo‘lishi
- `GSAP` (optional) – professional animatsion kutubxona

---

## ⚡ Asosiy xususiyatlar

- 🔥 React komponentlar asosida SPA (Single Page Application)
- 📱 Mobil va planshetga mos responsive dizayn
- 🎥 Smooth transition’lar va hover effektlar
- 🧠 Kod strukturasi toza va oson tushunarli
- 🎨 Ruxsatli Dark/Light rejim
- 🌐 Tilni almashtirish imkoniyati (UZ | RU | EN)
